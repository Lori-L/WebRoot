Web Resources

W3C Validator | web.dev | CodeCademy | YouTube | FreeCodeCamp | Main

  1. W3C Validator
    The W3C validator checks the formatting of a website to ensure that it meets W3C standards. If any issues are present, it locates and describes them, along with why they are issues in the first place.
    The W3C validator is a great resource for anybody new to HTML. It allows for quick and concise feedback, which is incredibly helpful for anyone who is still becoming familiar with HTML standards and best practices.
    Rating: 5 / 5

  2. web.dev
    The web.dev site is an instructional website put together by Chrome developers. There are many courses that help walk you through different aspects of web development.
    The one I'm reading through right now is for responsive design. Although I'm still earlier on in the course, it has already explained and clarified some things I was confused on. The explanations are good, and it also walks through example code while explaining each step, which has been very helpful to me so far.
    Rating: 5 / 5

  3. CodeCademy
    CodeCademy provides courses in a variety of languages at directed towards varying proficiency levels.
    The courses themselves are informative, allowing you to play with the code and see its results immediately. Tips and hints are given based on the type of mistakes made, which help prevent any missing backslash-induced frustations from growing. Learning is also gamified, with each day of engagement being measured, and multi-day streaks celebrated. Unfortunately, though, most content (especially the higher-level content) on this website is locked behind CodeCademy Pro, which does have a subscription fee.
    Rating: 4 / 5

  4. YouTube
    On YouTube, anybody can create and upload videos for others to see. Tips and tutorials are easy to find for a variety of different skills, as long as you know what you're looking for.
    Some fantastic resources can be found on YouTube, if you are willing to dig for a little bit. The difficulty comes in differentiating the well-made tutorials from the inaccurate and unhelpful ones - many videos claim to hold life-changing information, but are only trying to get more views. This can lead to wasted time that could have been spent learning and growing. Here are some good resources that I have found especially helpful:

    • Programming with Mosh
    • Web Dev Simplified
    • Video: How To Think Like A Programmer

    Rating: 3.5 / 5

  5. FreeCodeCamp
    FreeCodeCamp has thousands of coding tutorials spanning many different programming languages for, well, free!
    They also have a YouTube channel that uploads tutorials frequently. There are many informative articles, as well - for example, I refreshed my memory on how to make links open in a new tab using this article! It is difficult to stumble across an article that you haven't specifically searched for, though. This week, in an attempt to become more familiar with responsive web design, I started their Responsive Web Design Certification course. Although I am still very early in the course, the descriptions are thorough and easily understandable.
    Rating: 4.5 / 5
